Postav osobní web pro Petru Dujkovou, freelance marketingovou konzultantku z Frýdku-Místku. Web je v češtině, jednoduchý, přehledný a lidský. Texty a design systém jsou přiloženy v souborech.

## TECHNOLOGIE

- Čistý HTML + CSS + minimální vanilla JS
- Žádné frameworky
- Google Fonts: Syne (600, 700) + DM Sans (400, 500)
- Mobilní responsivita povinná
- Jeden soubor index.html + jeden style.css
